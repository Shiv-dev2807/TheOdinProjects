# Odin Recipes 🍽️

This is a basic HTML project built as part of The Odin Project curriculum.  
The goal of this project is to practice core HTML skills by building a simple recipe website.

## 📌 Project Overview

The website contains:
- A main homepage (`index.html`)
- Multiple recipe pages inside a `recipes/` folder
- Links between pages for navigation
- Images, headings, lists, and structured content using HTML

## 🧠 What I Learned

Through this project, I practiced:
- Creating HTML boilerplate structure
- Using headings, paragraphs, lists, and links
- Organizing files and folders properly
- Linking pages using relative paths
- Adding images to web pages
- Using Git and GitHub for version control
